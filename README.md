# FinScope Analytics

FinScope Analytics is a data analytics project focused on analyzing financial data to identify customer behavior, trends, and credit risks.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI

## Project Files
- Jupyter Notebook for data analysis
- Banking dataset
- Power BI dashboard
- Project presentation

## Key Insights
- Performed data cleaning and preprocessing
- Conducted exploratory data analysis
- Identified customer trends and credit risks
- Generated insights to support business decision-making
